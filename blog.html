What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a way for web browsers to understand and interact with an HTML or XML document. It organizes the content of a webpage into a tree-like structure, where each part (like tags, text, and attributes) is a "node." JavaScript uses the DOM to make changes to a webpage, such as updating text, changing styles, or responding to user actions like clicks. This allows web pages to be dynamic and interactive instead of static.

How do you select an element from the DOM?

To select an element from the DOM, you can use JavaScript methods like getElementById() to find an element by its ID, getElementsByClassName() to find elements by their class, or getElementsByTagName() to find elements by their tag name. For more flexibility, you can use querySelector() to select the first element that matches a CSS selector or querySelectorAll() to select all matching elements. These methods help you locate and interact with specific parts of a webpage.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events on its child elements. This works because events bubble up from the target element to its parent. How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes, you can use methods like setAttribute() to add or update an attribute, getAttribute() to retrieve an attribute's value, and removeAttribute() to remove an attribute. For styles, you can modify the style property directly, like element.style.color = "red";, or add/remove classes using classList, such as element.classList.add("myClass") or element.classList.remove("myClass"). These methods allow dynamic updates to an element's appearance and behavior.
